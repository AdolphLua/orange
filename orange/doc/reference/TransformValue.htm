Value Transformers

Class TransformValue is a base class for a small hierarchy of tiny classes that are use throughout Orange at the places where a simple transformation of values is needed. Some of such occasions are

  • discretization, where the class Discretizer defines a base class for a hierarchy of classes that turn continuous values into discrete by checking thresholds or interval cutoffs
  • SVM preprocessing, where Discrete2Continuous and NormalizeContinuous change discrete and continuous values into values from interval [0.0, 1.0]
  • decision trees with binarization (or subsetting), where MapIntValue computes a branch index from a discrete value.

Various classes derived from TransformValue are covered on pages with the corresponding topic.

It is possible to subtype this class in Python. An example of that is given in examples for Classifiers from Attribute.
