Orange Documentation

If you are new to Orange, we recommend you to start with Orange for Beginners, a step-by-step tutorial through basic Orange functions.

A more in-depth documentation that is being prepared is called Orange Reference Guide. It describes the basic data structures used in Orange (such as examples, attributes...), preprocessing, learning and classifying etc.

An average user will mostly call Orange through specialized Orange modules. Each module covers a specific task (discretization, feature subset selection, sampling, assessing qualities of models) or learning method.
