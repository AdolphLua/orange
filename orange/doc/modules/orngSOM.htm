orngSOM

orngSOM provides orange with SOM functionality

SOMLearner

SOMLearner is a function that constructs a SOMLearner instance and optionally trains it on provided examples

Arguments

examples
Examples to be used for training (default None)
wightID
Index of the meta attribute that contains weight values (default 0)
xDim
X dimension of the map (default 10)
yDim
Y dimension of the map (default 10)
topology
Topology of the map. Can be a SOMLearner.RetangularTopology for rectangular or SOMLearner.HexagonalTopology (default) for hexagonal topology
neighborhood
Neighborhood function type. Can be SOMLearner.BubbleNeighborhood (default) or SOMLearner.GaussianNeighborhood
alphaType
A alpha function type. Can be a SOMLearner.LinearFunction (default) or SOMLearner.InverseFunction
parameters
Specifies the radius, alpha and num. of iterations for each phase of training in a form of a list of dictionaries (e.g. parameters=[{"iterations":1000, "radius":5.0, "alpha":0.05}, {"iterations":10000, "radius":3.0, "alpha":0.02}] (note that not all (or even any) items in a dictionary are mandatory {"iterations:1000} is a valid item in the parameters list, the rest will be assigned the default or the last given value)

getUMatrix

getUMatrix is a function that constructs a u-matrix (uniform distance matrix) of the provided SOM (the way it is constructed depends on the topology of the SOM)

Arguments

somMap
SOM for witch the u-matrix is to be constructed

Examples
