• Orange and SQL

    There are currently four ways to access data, stored in a relational database, from Orange.

    • orngMySQL, the oldest module allowing access to MySQL only. No further development planned.
    • orngSQL, the new module, allowing access to any database which supports the standard python API.
    • The SQL widget, which is based on orngSQL
    • orngSQLFile, by opening .sql files containing SQL statements with special headers. Probably the best choice for most users.
