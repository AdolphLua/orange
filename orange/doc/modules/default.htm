Orange Modules


Orange modules are intended to extend basic Orange's functionality, or provide wrappers for easier use of some data mining techniques. They are already included in normal Orange distribution. Following set of pages provides description, demos and examples for selected modules. Following modules are included [for links that do not work, documentation is in writing - sorry]:

orngAssoc
Wrapper around Orange's association rules generator (builds association rules, prints them out).
orngCI
Constructive induction (function decomposition methods, HINT, Kramer's constructive induction method).
orngDisc
Wrapper around Orange's categorization techniques for continuous attributes.
orngEnsemble
Bagging, boosting.
orngEval
Obsolete, included for compatibility with past version of Orange. Use orngTest and orngStat instead.
orngFSS
Feature subset selection.
orngLookup
Functions for working with classifiers with stored tables of examples.
orngStat
This module takes an object that stores the results of evaluation of learning methods prepared by orngTest, and computes various statistics including accuracy, sensitivity, specificity, information gain, area under ROC, ...
orngTest
A module for testing of the learning algorithms. Includes methods for data sampling and splitting, and for testing of learners. Among others, it implements cross-validation, leave-one out, random sampling, learning curves. It prepares an object that stores the results on test data and that is used in orngStat to assess various statistics.
orngTree
Wrapper around Orange's classification tree inducer. Most notably, implements tree printout and visualization.

Additional modules for data mining (clustering, SVM wrapper for probability estimation, logistic regression, density estimation...) have been prepared by Aleks Jakulin and are accessible via his pages.
